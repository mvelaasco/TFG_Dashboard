# TFG Finance Frontend

Frontend básico en React + Vite para:

- listar activos disponibles
- lanzar el cálculo de correlaciones
- consultar y visualizar la serie de correlación

## Ejecutar

```bash
cd frontend
npm install
npm run dev
```

Por defecto el front apunta a `http://localhost:8000`.
Si tu API está en otra URL, crea un `.env` con:

```bash
VITE_API_BASE_URL=http://localhost:8000
```

## Build

```bash
cd frontend
npm run build
```
